# Hello World Python

A simple Python "Hello World" script demonstrating basic Python programming concepts.

## Description

This project contains a basic Python script that:
- Prints a personalized greeting
- Demonstrates Python lists and loops
- Shows basic string formatting
- Includes proper Python documentation

## How to Run

Make sure you have Python 3 installed, then run:

```bash
python3 hello_world.py
```

## Requirements

- Python 3.6+ (for f-string support)
